## 📌 Project Description
This project is developed as part of an internship task. It automates the process of reading data from a CSV file, performing analysis, and generating a well-structured PDF report.

The system helps in reducing manual effort by automatically analyzing data and presenting it in a clear and formatted report.

---

## ⚙️ Technologies Used
- Python
- Pandas (for data analysis)
- ReportLab (for PDF generation)

---

## 📁 Project Files
- input.csv → Contains student data
- report.py → Python script for processing and report generation
- report.pdf → Final generated report

---

## 🚀 How It Works
1. The script reads data from the CSV file.
2. It calculates:
   - Total number of students
   - Average marks
   - Highest and lowest marks
   - Subject-wise average
3. It generates a formatted PDF report containing:
   - Summary
   - Student data table
   - Subject-wise analysis
